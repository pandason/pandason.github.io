# dependency install command

```bash
# bundle: 루비 의존성 관리도구 bundler 를 이용하여 의존성 모듈 설치
$ bundle
```

# local init command

```bash
# jekyll: 정적 웹사이트 도구
# serve: Jekyll 의 로컬 개발 서버 시작 명령어
$ jekyll serve
```
